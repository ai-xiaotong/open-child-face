# OpenChildFace数据集

目前，商用人脸识别系统可以在大人数据集上达到非常高的识别准确率。
然而，在儿童人脸数据上，他们往往表现得不尽人意。
为了方便人脸识别在评估低年龄段的效果，我们从glint-asian and glint-msra中提取一个儿童人脸子集OpenChildFace，
包含45个ID和784张图像。

这个数据集可以从[百度云盘](https://pan.baidu.com/s/1gjmYh-Oo1p4zKWDdcYduGA)(提取码: gl08)下载。
